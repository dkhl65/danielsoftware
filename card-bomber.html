Card Bomber App

This is a simple Android game app based on Minesweeper and Voltorb Flip. It suppports English, French, Simplified Chinese and Traditional Chinese. The language is set based on your device settings. This app was written in Java using Android Studio.

The game's objective is to flip over all the cards without flipping a bomb. Clues indicating the value of cards and number of bombs in each row and column is provided. Your high score is saved on the device

Installation Instructions for Android

  1. On your Android device, enable app installation from unknown sources in system settings
  2. Click on the DOWNLOAD button and wait for the download to begin
  3. Save Card Bomber.apk to a folder on your device
  4. Browse to the downloaded APK file
  5. Click on the file and select the option to install the app
