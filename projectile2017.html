Projectile 2017

This is a projectile motion simulator. A cannon ball is animated in real time and view live values for the projectile's speed, location and angle. It can be used to solve Grade 12 physics problems involving initial speed, angle and final distance. Have fun challenging yourself to hit the target and finish your kinematics homework! This software was written in Java.

Instructions for Windows

  1. Click on the DOWNLOAD EXE button and wait for the download to begin
  2. Save Projectile 2017 Release.zip to your computer
  3. Right click the file and select "Extract all..." to extract the files
  4. Read User Manual.docx for instructions on how to use the program
  5. Make sure to have the latest version of Java installed
  6. Open Projectile2017.exe which is inside the Projectile 2017 folder

Instructions for Linux

  1. Click on the SOURCE CODE button to see the GitHub page
  2. Save Projectile2017.jar to your computer
  3. Make sure to have Java (e.g. OpenJRE) installed
  4. Read User Manual.docx for instructions on how to use the program
  5. The JAR file can be executed using Java Runtime Environment or on the command line: java -jar Projectile2017.jar
